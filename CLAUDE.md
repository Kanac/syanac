# syanac-website — Syanac studio marketing site

Static one-page marketing site for the **Syanac** studio (the app fleet lives in `../syanac-apps`).
Repo `Kanac/syanac`, served at **https://kanac.github.io/syanac/** (GitHub Pages). Just `index.html` +
`assets/` — **no build step**: edit and push, Pages redeploys. ⚠️ Don't rename the repo or change the Pages
path — the apps' store listings and the legal page reference this domain.
